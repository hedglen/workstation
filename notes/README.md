# Notes

Personal markdown notes (tech, workflow, general). This folder lives **inside the [workstation repo](https://github.com/hedglen/workstation)** at `notes/` (`notes/personal/` is gitignored and never pushed) — commit with the rest of your workstation config, or keep sensitive paths out of the public remote via branch/private fork as you prefer.

Machine layout, apps, and bootstrap: **`docs/workstation-setup.md`**.

## Structure

```text
notes/
├── personal/   personal notes
├── tech/       technical references and how-tos
└── work/       work-related notes
```
